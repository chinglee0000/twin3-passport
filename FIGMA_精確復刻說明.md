### CSS 定位
- 使用 `position: absolute` 精確定位所有元素
- 所有尺寸都按照 Figma 的像素值設定

### 響應式設計
- 主要針對 1440px 寬度設計
- 小螢幕使用 CSS `transform: scale()` 縮放
- 保持原始比例和佈局

### 字體載入

- 精確匹配字重和樣式
